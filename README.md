# 📚 Data Structures & Algorithms Practice

Welcome to my DSA practice repository! 🚀

This repository contains my journey of learning and solving Data Structures and Algorithms problems using **C#**.

* Strengthen problem-solving skills
* Improve coding interview preparation
* Learn common DSA patterns
* Write clean and optimized solutions
* Track my progress consistently

## 🧠 Problem Solving Approach

For each problem I try to:

1. Understand the problem.
2. Build a brute-force solution.
3. Analyze time and space complexity.
4. Optimize the solution.
5. Document key learnings.
